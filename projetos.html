Jogo de Escolha de Número

Este é um jogo simples de adivinhação criado usando HTML, CSS e JS. O jogador é desafiado a adivinhar um número entre 0 e 25. O jogo apresenta um formulário com um campo de entrada para o número e um botão para submeter a resposta. O jogador pode tentar várias vezes e o número de tentativas é exibido em uma seção de resultados. Além disso, há um botão "Jogar novamente" que permite ao jogador reiniciar o jogo.

Login-Helper

Este é um formulário de login criado em HTML, CSS e JavaScript. Ele contém campos de entrada de email e senha. O campo de senha é ocultado por asteriscos por padrão, mas há um botão com um ícone de olho que permite a visualização da senha. Há também uma caixa de seleção para "Lembrar-me" e um link para "Esqueceu sua senha?". Existem três ícones para login com Google+, LinkedIn e Facebook.

Efeito Parallax
